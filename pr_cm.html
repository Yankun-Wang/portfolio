Intro

Problem Roulette

Problem Roulette (PR) is a point-free study service that offers topical access to a library of locally-authored exam problems in selected foundational courses at the University of Michigan.

Problem Overview

Building a more dynamic platform with gamification elements

Currently, Problem Roulette has two modes for users to study, the Study Mode and the Exam Mode. The product team was seeking an opportunity to add a new mode to the platform and make it more dynamic. After discussing with the behavior scientists, we decided to add some gamification elements to the platform, which ended up being the Challenge Mode. My task was to research what users expect from this new mode and design its user experience.
